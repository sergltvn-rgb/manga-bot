# Инструкция по запуску WebApp (Веб-приложения для Telegram)

Веб-приложение (папка `webapp`) — это мини-сайт, который открывается прямо внутри Telegram. Чтобы кнопка "🌐 Веб-чат с Алей" заработала у всех пользователей, этот сайт нужно разместить в интернете (на хостинге).

## Шаг 1: Настройка API ключа в `app.js`
В файле `webapp/app.js` (строка ~82) есть переменная:
`const API_KEY = "СЮДА_НУЖНО_ВСТАВИТЬ_КЛЮЧ";`

Для того чтобы чат работал, вам нужно:
1. Скопировать ваш `GROQ_API_KEY` из файла `codes.env`.
2. Вставить его вместо текста `"СЮДА_НУЖНО_ВСТАВИТЬ_КЛЮЧ"`.

⚠️ **Важно:** В реальных (крупных) проектах API ключи не хранят в `app.js`, так как их могут украсть из кода браузера. Но для старта и личного использования этого достаточно. Помните, что ключ Groq бесплатный, но имеет лимиты.

## Шаг 2: Бесплатный хостинг сайта (через GitHub Pages)
Самый простой бесплатный способ разместить ваш сайт:
1. Зарегистрируйтесь на [GitHub](https://github.com/).
2. Скачайте приложение [GitHub Desktop](https://desktop.github.com/) или используйте сайт.
3. Создайте новый публичный репозиторий (например, `manga-bot-webapp`).
4. Загрузите туда файлы `index.html`, `app.js`, `style.css` (содержимое папки `webapp`).
5. Зайдите в **Settings** (Настройки репозитория) -> вкладка **Pages** (слева).
6. В разделе **Build and deployment** -> Source выберите `Deploy from a branch`.
7. Выберите ветку `main` (или `master`) и папку `/ (root)`. Нажмите `Save`.
8. Через пару минут вверху появится ссылка вида: `https://ваш-ник.github.io/manga-bot-webapp/`.

## Шаг 3: Подключение сайта к боту
1. Откройте файл `codes.env` (где лежат ваши токены).
2. Добавьте новую строку с вашей ссылкой на сайт (которую вы получили на Шаге 2). Пример:
   `WEBAPP_URL="https://ваш-ник.github.io/manga-bot-webapp/"`
3. Перезапустите бота (скрипт `run.bat`).
4. Зайдите в бота, напишите `/start` и нажмите на кнопку **"🌐 Веб-чат с Алей"**. 

Всё готово! Приложение откроется прямо в Telegram.
