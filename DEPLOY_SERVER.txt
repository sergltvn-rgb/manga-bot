# Команды для выгрузки кода на сервер (CMD)

Так как у тебя уже инициализирован Git и подключен GitHub (`https://github.com/sergltvn-rgb/manga-bot.git`), самый правильный способ — это использовать Git.

## Способ 1: Через GitHub (Рекомендуется)
Этот способ сохраняет историю изменений и позволяет легко обновлять бота.

### 1. Отправка изменений в GitHub (на твоем ПК):
Выполни эти команды в CMD в папке с ботом:
```cmd
git add .
git commit -m "Оптимизация кода и добавление WebApp"
git push origin main
```

### 2. Загрузка кода на сервере (на VPS):
Зайди на свой сервер через SSH и в папке бота выполни:
```bash
git pull origin main
```

---

## Способ 2: Прямая загрузка через SCP (без GitHub)
Если ты хочешь просто перекинуть файлы напрямую с рабочего стола на сервер одной командой.

### Шаблон команды:
```cmd
scp -r "C:\Users\litvi\OneDrive\Desktop\bot\*" пользователь@IP-СЕРВЕРА:/путь/к/папке/на/сервере
```
*Замени `пользователь`, `IP-СЕРВЕРА` и `/путь/...` на свои данные.*

---

## Способ 3: Команда для создания архива (если нужно)
Если ты хочешь быстро запаковать всё в один файл перед отправкой:
```cmd
powershell Compress-Archive -Path * -DestinationPath ..\bot_update.zip -Force
```

**Рекомендация:** Используй **Способ 1**. Это стандарт в разработке. Если на сервере еще нет кода, сначала клонируй его там командой:
`git clone https://github.com/sergltvn-rgb/manga-bot.git`
